# 🚀 Newpen Smart System: Automação, SQL & BI

Este sistema é uma solução completa de **Inteligência Comercial** desenvolvida integralmente por **William Estevam**. O projeto integra banco de dados relacional, automação de processos (RPA) e visualização de dados (Business Intelligence).

## 📋 Sobre o Projeto
O sistema foi projetado para escalar a operação comercial, substituindo o gerenciamento manual de planilhas por uma infraestrutura de dados moderna. A transição para **SQL (SQLite)** permite um controle preciso sobre múltiplas ondas de prospecção, garantindo que nenhum cliente seja impactado em duplicidade.

## ✨ Funcionalidades de Elite
* **Dashboard de BI:** Monitoramento do funil de vendas em tempo real com gráficos interativos.
* **Persistência em SQL:** Armazenamento seguro de contatos e status de envio.
* **Automação (RPA):** Robô inteligente para disparos personalizados via WhatsApp.
* **Leads Management:** Identificação automática de clientes interessados para ação imediata.

## 🛠️ Tech Stack
* **Linguagem:** Python 3.12+
* **Banco de Dados:** SQLite
* **Automação:** Selenium & Webdriver Manager
* **BI:** Streamlit & Plotly
* **Processamento:** Pandas
